\定義{1.2}

\begin{list}{}{}
	\item	(ⅰ) 環$A$の元$x$について，$xy=0$となる$y \ne 0$が存在するとき，$x$を\textbf{零因子}（zero divisor）という
（すなわち，$\displaystyle y = \frac{0}{x}$；$0$を割って$y \ne 0$が出る）．$0$でない零因子を持たない環$(\ne 0)$を
\textbf{整域}（integral domain）という．
	\item (ⅱ) ある自然数$n$に対し$x^n = 0$となる元を\textbf{ベキ零元}（nilpotent element）というベキ零元は零因子である．
	\item (ⅲ) 乗法について逆元をもつもの，すなわち，$x \in A$で$xy = 1$となる$y \in A$が存在するとき$x$を\textbf{単元}（unit）といい，$y=x^{-1}$，
単元の集合を$A^{\times}$と書く．$A^\times$は空集合でなければ$1$を単位元とする群をなし，$A$の\textbf{単元群}（unit group）という．
	\item (ⅳ) $A^\times = A \setminus \{ 0 \} \ne \varnothing$のとき$A$を\textbf{（可換）体}
（commutative field）という．
\end{list}
\□

有理整数環$\Z$は整域であり名前もそこに由来する．体も整域で，$a \ne 0$ならば$a x = b$をみたす$x \in A$が
唯一存在する$(x = a^{-1} b)$．$\Z$の単元は$\pm 1$である．
